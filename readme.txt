=== Advanced Custom Toolkit ===

Contributors: Abubakar
Version: 1.0.0

== Description ==
A professional WordPress plugin with shortcode, admin settings, and AJAX-based post view tracking.

== Features ==
- Shortcode support
- Admin settings panel
- AJAX view tracking
- Session-based view protection
- Clean OOP structure

== Installation ==
1. Upload plugin folder
2. Activate plugin
3. Use shortcode: [custom_message text="Hello"]
